While e Do/While

O while e o do/while são dois operadores utilizados para iteração.

Sua grande diferença em relação ao laço for, é que ele somente possui a condição de parada em sua declaração, deixando a inicialização e incremento de valores para acontecerem antes ou durante o comando de iteração.

O while possui uma variação chamada do/while, na qual a diferença ocorre na verificação da condição de parada do laço, uma vez que ela ocorrerá no início para o comando while, e no final para o comando do/while.

While

Vamos iniciar com um exemplo do laço while:


int numero = 1;
boolean fim = false;

while (!fim) {
  System.out.println("Ainda não terminou...");
  fim = true;
}

while (numero <= 10) {
  System.out.println(numero);
  numero++;
}

No código acima, criamos duas variáveis, "numero" e "fim", que conterão os valores de um contador e a informação se deve terminar a execução do laço, respectivamente.

Logo em seguida, temos o primeiro laço while, que será executado enquanto o valor de "fim" for diferente de "true". No caso de código, ele será executado uma única vez, visto que o valor de "fim" é modificado para "true" já na primeira execução do laço.

No segundo laço, temos um while que será executado enquanto o valor de "numero" for menor ou igual a 10. Repare que o valor de "numero" é incrementado em um a cada execução do laço.

Inicialização e incremento de valores

O segundo comando while do código acima deixa bastante claro a inicialização e incremento de valores, que não ocorrem na definição do comando de laço, mas sim antes e durante sua execução.

A inicialização no mesmo código foi realizada anteriormente, quando definimos "numero" com o valor "1", e o incremento é feito com o operador de incremento "++", dentro do próprio laço.

Operador de incremento

Você certamente percebeu o uso do operador "++" para incrementar o valor da de "numero" no código acima.

Esse operador é uma alternativa a seguinte escrita:


numero = numero + 1; // ou
numero += 1;

O mesmo código é válido para o operador de decremento, que no caso seria "--".

Do/While

Vamos agora analisar um exemplo do laço do/while:


int numero = 1;

do {
  System.out.println(numero);
  numero++;
} while (numero <= 10);

No código acima, criamos a variável "numero", que contem o valor inicial do nosso contador, que será utilizado para terminar a execução do laço.

Logo em seguida, temos o laço do/while, que será executado enquanto o valor de "numero" for menor ou igual a "10".

A diferença desse código para o estudado na página anterior, é que a condição é verificada no final da execução do laço, e não no início como o while.

Quando usar um ou outro?

A escolha de qual usar dependerá muito do seu algoritmo, pois ambos trabalham de modo muito similar, mas na prática você preceberá que na maioria dos casos o while por si só atenderá a quase todos os casos, sendo o mais utilizado.
