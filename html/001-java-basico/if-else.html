Entendo o If/Else

1. Na criação de qualquer código, necessitamos utilizar a todo tempo condicionais para direcionar as etapas de execução de nosso algoritmo.

2. Para isso existe o If e Else, para nos permitir executar um determinado bloco de código dependendo se uma condição é válida.

3. A condicional a ser executada pode conter diferentes variações, vamos estudar a seguir as principais delas.

Criando o primero programa - "Olá mundo!"

1. A mais simples delas é o uso apenas do if, em conjunto com uma única condição, conforme o código a seguir:


boolean passouNoTeste = true;
int nota = 80;

if (passouNoTeste) {
  System.out.println("Passou no teste!");
}

if (!passouNoTeste) {
  System.out.println("Não passou no teste.");
}

if (nota >= 70)
  System.out.println("Passou no teste!");

2. Repare que usamos a palavra chave if seguida da condição, que estará entre parênteses.

3. A condição deverá sempre retornar um valor booleano, para que o código contido entre chaves seja ou não executado.

4. O operador "!" significa negação, ou seja, caso o resultado da condição seja verdadeira, ele será convertido para falso, e vice versa.

5. Podemos também utilizar operadores de comparação, que são os "==", "<", ">", "≥" e "≤", onde comparamos se os valores são iguais, menores, maiores, maiores ou igual, e menores ou igual, respectivamente.

Uso das chaves

Quando temos uma única linha de código em nosso bloco if, podemos simplesmente omitir as chaves.

Condicionais compostas

1. Com o uso dos operadores "e", representado por &&, e do operador "ou", representado por ||, podemos encadear múltiplas condições em um mesmo if, conforme o código a seguir:


boolean passouNoTeste = true;
int nota = 80;

if (nota >= 70 && nota <= 80) {
  System.out.println("Passou no teste com nota B.");
}

if (passouNoTeste || nota >= 70) {
  System.out.println("Passou no teste!");
}

2. Basicamente o operador "e" fará com que o bloco de código seja executado, somente se ambas condições forem verdadeiras, enquanto que o operador "ou" será executado se ao menos uma das condições for verdadeira.

Múltiplas condicionais

Embora não exista um limite de condições, adicionar muitas delas em um mesmo if tornará o código de difícil legibilidade, portanto o recomendado é limitar ao máximo o número de condições em sequência.

If e Else em conjunto

1. Muitas vezes queremos não somente executar o código para o resultado verdadeiro de uma condição, mas sim também para quando o resultado for falso, e para isso utilizamos o else, conforme o código abaixo:


boolean passouNoTeste = true;
int nota = 80;

if (passouNoTeste) {
  System.out.println("Passou no teste!");
} else {
  System.out.println("Não passou no teste.");
}

if (nota % 2 == 0) {
  System.out.println("Nota é um número par.");
} else {
  System.out.println("Nota é um número ímpar.");
}

if (nota < 70) {
  System.out.println("Não passou no teste.");
} else if (nota >= 70 && nota <= 90) {
  System.out.println("Passou no teste com nota B.");
} else {
  System.out.println("Passou no teste com nota A.");
}

2. No primeiro e segundo if do código acima, executamos uma ação para quando o resultado for verdadeiro, e outra para quando ele for falso (else).

3. No terceiro if do código acima, verificamos não somente duas, mas três condições distintas, combinando o else com o if, onde poderemos verificar quantas condições forem necessárias.

Operador "%"

O operador "%", ou módulo como é chamado, retorna o restante de uma divisão. Um clássico exemplo é testar se um número é par ou ímpar, conforme o código acima, onde ao dividirmos um número por 2, caso o restante seja 0, significa que ele é par, e para qualquer outro valor ele será ímpar.

Múltiplos ifs e elses

Encadear múltiplas condições nem sempre é uma boa ideia, e muitas vezes é melhor utilizar o operador switch para isso, que estudaremos em breve.
