Instalando o Java

O Java necessita ser executado em uma máquina virtual, a JVM (Java Virtual Machine).

Para o desenvolvimento de aplicações Java, precisamos instalar a JDK (Java Development Kit), que nos permite compilar e executar códigos Java.

Para instalar a JDK, devemos realizar o download aqui.

Acessando a URL acima, faça o download referênte ao sistema operacional e execute o instalador.

Para verificar a sua instação, execute o seguinte comando no terminal:


java -version

Você deverá ver algo semelhante a:


java version "1.8.0_45"
Java(TM) SE Runtime Environment (build 1.8.0_45-b14)
Java HotSpot(TM) 64-Bit Server VM (build 25.45-b02, mixed mode)

Criando o primero programa - "Olá mundo!"

Primeiramente, crie em algum diretório um arquivo chamado OlaMundo.java, com o seguinte conteúdo:


public class OlaMundo {
  public static void main(String[] args) {
    System.out.println("Olá mundo!!!");
  }
}

Após a criação do arquivo, devemos compilar o mesmo com o seguinte comando (executado pelo terminal na raiz do diretório onde você criou o arquivo):


javac OlaMundo.java

Após a compilação, o arquivo OlaMundo.class será gerado no mesmo diretório.

Para executar o código, execute o seguinte comando na raiz do diretório onde você criou o arquivo Java:


java OlaMundo

Você deverá vizualizar o seguinte:


Olá mundo!!!

Utilizando uma IDE

No curso utilizaremos apenas a linha de comando para criar os arquivos, mas você poderá utilizar uma IDE para facilitar o trabalho, para isso você poderá baixa o IntelliJ ou o Eclipse por exemplo.

Entendendo o código do programa - "Olá mundo!"

Vamos começar analisando a linha da declaração da classe:


public class OlaMundo {
}

O public significa que a classe pode ser acessada de qualquer local, ela é pública.

Escopo das classes

Existem outros escopos de classe, como o protected, private e default, que serão apresentados futuramente, por hora vamos utilizar somente o public.

O class significa que esse arquivo é uma classe, que será utilizado para criar objetos (instâncias) em Java, e toda classe deve possuir um nome, no nosso caso OlaMundo.

Nomes das classes

Existe uma convenção na nomenclatura de classes, que basicamente é a primeira letra sempre maiúscula, e para cada nova palavra usamos sempre a primeira letra maiúscula também. Não devemos utilizar espaços nem caracteres especiais em seu nome.

Nomes dos arquivos

Todos os arquivos deverão ser nomeados com o mesmo nome definido na classe, mais a extensão .java.

Logo na sequência temos o método principal, main, que é o método padrão em classes Java a ser executado sempre que a executamos, conforme o código a seguir:


public static void main(String[] args) {
}

Nesse método a declaração public significa que o método é público, ou seja, pode acessado de qualquer local, visível por qualquer outra classe que tenha acesso a uma instância da classe.

O static significa que o método é estático, ou seja, que ele pode acessado sem que tenhamos uma instância da classe, é um operador que estudaremos em maiores detalhes mais adiante.

O void significa que o método não retorna nenhum valor.

O String[] args significa que o método main ao ser executado, caso sejam passados valores pela linha de comando, as informações estarão disponíveis nessa lista de Strings.

Para finalizar utilizamos o comando padrão para imprimir uma linha no terminal, conforme o código a seguir:


System.out.println("Olá mundo!!!");
