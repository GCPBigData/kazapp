Switch

O switch é um comando utilizado para executar uma determinada ação de acordo com o valor passado a ele para verificação.

Ele recebe um único parâmetro como entrada, e para este valor ele verifica se existe uma condição (case) definida nele.

O switch pode possuir inúmeros "cases", assim como uma codição padrão (default), que será executada caso a entrada não atenda a nenhum "case".

O switch é bastante similar ao if/else, mas sua grande vantagem é que sua legibilidade é melhor para quando queremos fazer diversas verificações.

Sintaxe do switch

Verifique no código a seguir a definição do comando switch:


switch(VALOR_A_SER_VERIFICADO) {
   // o VALOR_A_SER_VERIFICADO deve ser do mesmo tipo do validado a seguir
   case VALOR_1:
      // seu código para o VALOR_1
      break; // opcional, interrompe a execução do switch
   
   case VALOR_2:
      // seu código para o VALOR_2
      break; // opcional, interrompe a execução do switch
   
   default : 
      // código padrão para quando nenhuma condição for válida
}

O switch recebe um valor que será utilizado nas verificações das possibilidades (case).

Dentro do switch temos as definições das condições (case), onde verificamos cada um dos possíveis valores.

Utilizamos o break, que é opcional para terminar a execução para cada caso.

Podemos também adicionar a entrada default, que é um case especial para quando nenhuma condição for encontrada.

O switch na prática

Vamos primeiramente estudar o uso do switch para retornar o nome de um mês dado um valor numérico entre 1 e 12.

Para qualquer outro valor retornamos que o mês é inválido.

Veja a seguir o código:


int mes = 7;
String nomeMes;

switch (mes) {
  case 1:
    nomeMes = "Janeiro";
    break;
  case 2:
    nomeMes = "Fevereiro";
    break;
  case 3:
    nomeMes = "Março";
    break;
  case 4:
    nomeMes = "Abril";
    break;
  case 5:
    nomeMes = "Maio";
    break;
  case 6:
    nomeMes = "Junho";
    break;
  case 7:
    nomeMes = "Julho";
    break;
  case 8:
    nomeMes = "Agosto";
    break;
  case 9:
    nomeMes = "Setembro";
    break;
  case 10:
    nomeMes = "Outubro";
    break;
  case 11:
    nomeMes = "Novembro";
    break;
  case 12:
    nomeMes = "Dezembro";
    break;
  default:
    nomeMes = "Mês inválido";
    break;
}
System.out.println(nomeMes);

No código anterior imprimimos o valor do mês 7, que no caso é julho.

Sintaxe do switch, segundo exemplo

Nesse segundo exemplo, vamos imprimir todos os nomes dos meses a partir do mês passado no comando switch.

Segue o código do exemplo:


int mes = 7;

switch (mes) {
  case 1:  System.out.println("Janeiro");
  case 2:  System.out.println("Fevereiro");
  case 3:  System.out.println("Março");
  case 4:  System.out.println("Abril");
  case 5:  System.out.println("Maio");
  case 6:  System.out.println("Junho");
  case 7:  System.out.println("Julho");
  case 8:  System.out.println("Agosto");
  case 9:  System.out.println("Setembro");
  case 10: System.out.println("Outubro");
  case 11: System.out.println("Novembro");
  case 12: System.out.println("Dezembro");
           break;
  default: break;
}

Repare no uso do break somente no final do case 12, pois assim as condições serão executadas sequencialmente a partir do mês passado como parâmetro.

Caso seja passado um valor diferente de 1 a 12, nada será impresso, pois a condição default não executando nenhum comando.

Uso do "default"

Embora o comando default seja opcional, é sempre recomendado sua utilização, pois assim teremos a certeza de que trataremos todas as possíveis condições do nosso switch.

Sintaxe do switch, terceiro exemplo

No terceiro exemplo, vamos imprimir o número de dias de um determinado mês.

Usaremos case(s) agrupados para representar os meses com 30 dias, 31 dias, e para o caso particular de fevereiro, que pode conter 28 ou 29 dias.

Verifique o código a seguir:


int mes = 7;
int ano = 2018;
int numDias = 0;

switch (mes) {
  case 1: case 3: case 5: case 7: case 8: case 10: case 12:
    numDias = 31;
    break;
  case 4: case 6: case 9: case 11:
    numDias = 30;
    break;
  case 2:
    if (((ano % 4 == 0) && !(ano % 100 == 0)) || (ano % 400 == 0)) {
      numDias = 29;
    } else {
      numDias = 28;
    }
    break;
  default:
    System.out.println("Mês inválido.");
    break;
}
System.out.println("Número de dias = " + numDias);

Repare que agrupamos os case(s) simplesmente adicionando a entrada case, uma seguindo a outra, sem nenhuma implementação, até que a última possua o código para tratar todos os casos anteriores, seguido do break para terminar a execução do comando.

Esse comportamento apresentado é muito similar ao if utilizando condições compostas com o "ou" (||).

O exemplo acima imprimirá "Número de dias = 31", pois é a quantidade de dias do mês 7 (julho).

Vamos praticar?

A melhor forma de aprender é praticando, portanto vamos seguir um passo a passo para executar o código estudado nas aulas anteriores.

Crie o arquivo Switch.java em um diretório de sua preferência.

Adicione o seguinte código nele:


public class Switch {
  public static void main(String[] args) {
    int mes = 7;
    int ano = 2018;
    int numDias = 0;
    String nomeMes;

    switch (mes) {
      case 1:
        nomeMes = "Janeiro";
        break;
      case 2:
        nomeMes = "Fevereiro";
        break;
      case 3:
        nomeMes = "Março";
        break;
      case 4:
        nomeMes = "Abril";
        break;
      case 5:
        nomeMes = "Maio";
        break;
      case 6:
        nomeMes = "Junho";
        break;
      case 7:
        nomeMes = "Julho";
        break;
      case 8:
        nomeMes = "Agosto";
        break;
      case 9:
        nomeMes = "Setembro";
        break;
      case 10:
        nomeMes = "Outubro";
        break;
      case 11:
        nomeMes = "Novembro";
        break;
      case 12:
        nomeMes = "Dezembro";
        break;
      default:
        nomeMes = "Mês inválido";
        break;
    }
    System.out.println(nomeMes);

    switch (mes) {
      case 1:  System.out.println("Janeiro");
      case 2:  System.out.println("Fevereiro");
      case 3:  System.out.println("Março");
      case 4:  System.out.println("Abril");
      case 5:  System.out.println("Maio");
      case 6:  System.out.println("Junho");
      case 7:  System.out.println("Julho");
      case 8:  System.out.println("Agosto");
      case 9:  System.out.println("Setembro");
      case 10: System.out.println("Outubro");
      case 11: System.out.println("Novembro");
      case 12: System.out.println("Dezembro");
               break;
      default: break;
    }

    switch (mes) {
      case 1: case 3: case 5: case 7: case 8: case 10: case 12:
        numDias = 31;
        break;
      case 4: case 6: case 9: case 11:
        numDias = 30;
        break;
      case 2:
        if (((ano % 4 == 0) && !(ano % 100 == 0)) || (ano % 400 == 0)) {
          numDias = 29;
        } else {
          numDias = 28;
        }
        break;
      default:
        System.out.println("Mês inválido.");
        break;
    }
    System.out.println("Número de dias = " + numDias);
  }
}

Compile o código fonte, executando no terminal no diretório onde o arquivo foi criado:


javac Switch.java

O arquivo Switch.class será gerado no mesmo diretório.

Execute o código compilado no terminal com:


java Switch

Você deverá ver o seguinte no terminal:


Julho
Julho
Agosto
Setembro
Outubro
Novembro
Dezembro
Número de dias = 31

Parabéns!!! Você concluiu com sucesso o seu estudo do comando switch.
