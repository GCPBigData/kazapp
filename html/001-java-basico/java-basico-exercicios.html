Qual o comando utilizado para compilar programas Java pela linha de comando?

O que será impresso no terminal ao executar o código a seguir?


public class IfElse {
  public static void main(String[] args) {
    int nota = 80;

    if (nota >= 70 && nota <= 80) {
      System.out.println("Passou no teste com nota B.");
    } else {
      System.out.println("Passou no teste com nota diferente de B.");
    }
  }   
}

O que será impresso no terminal ao executar o código a seguir?


public class IfElse {
  public static void main(String[] args) {
    int nota = 79;

    if (nota % 2 == 0) {
      System.out.println("A linha do IF foi executada.");
    } else {
      System.out.println("A linha do ELSE foi executada.");
    }
  }   
}
